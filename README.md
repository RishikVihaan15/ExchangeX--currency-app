# 💱 ExchangeX

A modern currency dashboard built with **React**, **Vite**, and **Tailwind CSS** that lets users convert, compare, and track exchange rates across multiple currencies.

---

## ✨ Features

- 🌍 Real-time currency conversion
- 📈 7-day exchange rate trends
- ❤️ Favorite currencies
- 🕒 Conversion history
- 🔍 Compare multiple currencies
- 📱 Responsive design

---

## 🛠 Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript
- React Router
- PNPM

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/RishikVihaan15/ExchangeX--currency-app.git
```

### Install dependencies

```bash
pnpm install
```

### Start the development server

```bash
pnpm dev
```
